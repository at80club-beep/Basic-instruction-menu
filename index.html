🍳 泣いている子への対応

🧂 まずは準備すること

・ゆっくりした声

・しゃがんで目線を合わせる姿勢

・まずは安心させる気持ち

🔥 手順

1. 深呼吸して、落ち着いた声で名前を呼ぶ

2. しゃがんで目線を合わせる

3. 「大丈夫だよ」と短く安心させる

4. 落ち着いたら、何があったか一言だけ聞く

5. 小さな成功体験（できること）を一緒に作る

💡 コツ

・理由を長く聞かない

・泣き止ませることが目的ではなく“安心”が目的

・できたことを必ず褒める

困ったら、すぐ呼んでくださいね。

🍳 動かない子への対応

🧂 まずは準備すること

・待つ余裕

・小さな成功体験を作る気持ち

・焦らせない姿勢

🔥 手順

1. 無理に動かそうとせず、まずは近くに寄る

2. 「ここだけ一緒にやってみよう」と小さな一歩を提案

3. できたらすぐに褒める

4. 次の一歩を一緒に決める

💡 コツ

・「なんでできないの？」は絶対に言わない

・成功体験は“とても小さくていい”

・周りと比べない

あなたのペースで大丈夫ですよ。

🍳 けんか・トラブル対応

🧂 まずは準備すること

・落ち着いた声

・事実を確認する姿勢

・子どもを責めない気持ち

🔥 手順

1. まずは双方を落ち着かせる

2. 事実だけを短く確認する

3. どちらも否定せず、気持ちを言葉にしてあげる

4. 次にどうするかを一緒に決める

💡 コツ

・片方だけを信じない

・感情ではなく“事実”を扱う

・すぐに解決しなくてもOK

落ち着いていれば、必ずうまくいきます。

🍳 子どもが飽きてきたとき

🧂 まずは準備すること

・明るい声

・小さな変化を作る気持ち

・無理に引き戻さない姿勢

🔥 手順

1. 「ちょっとだけ変えてみようか」と声をかける

2. 道具・場所・役割などを少し変える

3. できたらすぐに褒める

4. それでも難しければ、短い休憩を提案

💡 コツ

・飽きるのは悪いことではない

・変化は“ほんの少し”で十分

・無理に戻そうとすると逆効果

気分転換も大事な時間ですよ。

🍳 保護者が見ているとき

🧂 まずは準備すること

・落ち着いた声

・子どもを主役にする姿勢

・保護者への安心感

🔥 手順

1. 子どもに向けて説明する（保護者ではなく）

2. 子どものできた部分を必ず言葉にする

3. 保護者には短く「大丈夫ですよ」と伝える

4. 子どもが自分でできた瞬間を一緒に喜ぶ

💡 コツ

・保護者の視線を気にしすぎない

・子どもを褒めると保護者も安心する

・説明は短く、行動はゆっくり

子どものペースを大切にすれば大丈夫です。

🍳 時間がない時

🧂 まずは準備すること

・優先順位を決める気持ち

・焦らない姿勢

・子どもを急かさない心

🔥 手順

1. まず深呼吸して落ち着く

2. 「絶対に必要な工程」だけを選ぶ

3. 子どもには「ここだけ一緒にやろう」と伝える

4. 終わったら必ず褒める

💡 コツ

・焦ると子どもは動かなくなる

・“削る工程”を決めておくと安心

・完成よりも“体験”を優先する

焦らなくても、ちゃんと進みますよ。
